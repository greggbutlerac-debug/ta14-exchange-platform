# TA-14 Exchange Web Application Scaffold

This directory reserves the first-release application boundary. The verified v0.2 package focuses on the domain, cryptographic receipts, AER manifest, database hardening, and acceptance tests before UI dependencies are introduced.

The first implemented screens will be: Start/Sign in, Organization setup, System registration, Route definition, Authority, Evidence, Test result, Correction, AER preview, Checkout, Record confirmation, Registry lookup, Account history, and Admin control.
