TA-14 GitHub Actions workflow setup

The .github/workflows folder was intentionally removed from this upload package because GitHub's browser uploader may freeze when dragging hidden dot-folders.

After the main project is uploaded:
1. In GitHub, click Add file -> Create new file.
2. In the filename box, enter exactly:
   .github/workflows/ci.yml
3. Open the separate TA14_CI_WORKFLOW_ci.yml file, copy all contents, and paste them into GitHub.
4. Commit directly to main.

Do not drag a .github folder into the browser uploader.
